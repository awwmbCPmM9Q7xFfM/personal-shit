Description

This project is meant to document certain details and characteristics of different countries. In this case I chose 25 different countries and show graphs of things such as GDP, Population, land area, phone usage, etc. and compare data with different countries.

Instructions

Click on the links of the website on "Project links" (use v14 for latest version) and use the nav bar to go through the country data sets.

How it works

The data sets, especially the graphs on the website, were made using python by processing files downloaded from the internet, with valid sources. These are .csv files which become lists, and then it is read to make graphs. The graphs are then displayed on web pages using HTML files. Also, some of the websites such as the ones on HTMLWeb and countries.html were made with python.

List of Files

The file AAFinal.py is the python program used to do all of the data processing, such as downloading/extracting the zip files, creating lists using the csv files, and making graphs using the data provided.

The file countryrandom.pyis a Python file that displays itself as a web page, as it will randomly change the country that is displayed when the page refreshes.

The file wikipedia-iso-country-codes.csv contains the data for the countries' names, 3 letter codes and 2 letter codes. We need this because the websites do not use the country name for downloading, but instead use the 3 letter or 2 letter codes to access the information.

all.html shows the data of all countries into 7 graphs.

country.htmlshows the lists of all countries to view their html files.

home.html shows the home page

style.css is the file used to customize the html files to give it a better look.

The (country).zip files are downloaded from the internet which contain the csv files needed

The (country).csv files contain all of the recorded data from the countries, and are used when making graphs.

The graphs on the /Graphs folder contain all of the .png files that are needed in order to present the information on the website.

The HTML files on /HTMLWeb/ are the HTML website files that show data for every country

Bugs and Side notes

Sometimes the graphs will cause a sudden climb, this is mostly because data before that period of time has not been recorded, so it goes from 0 to whatever large number.

If a graph is all 0 or does not have any data, it might be because there is no data on that topic.
